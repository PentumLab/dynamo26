DYNAMO-26 PIPELINE SPEC v1.0
Deterministischer Executor-Entwurf
Sprache: Deutsch
Ziel: Ein vollständig prüfbarer, quellengetriebener Tageslauf zur Bewertung von WM-2026-Titelstabilität.

============================================================
0. GRUNDPRINZIP
============================================================

Diese Spezifikation beschreibt keinen Fließtext-Prompt und keine lose Checkliste.
Sie beschreibt einen deterministischen Ablauf.

Jeder Schritt hat:
- Eingabe
- Pflichtaktion
- Rückgabeobjekt
- Gate
- Audit

Ein Schritt darf erst verlassen werden, wenn sein Rückgabeobjekt vollständig ist.
Ein späterer Schritt darf keine Daten erfinden, die ein früherer Schritt nicht geliefert hat.

============================================================
1. FESTE REGELN
============================================================

RULE_01:
Ohne aktuellen Quellenlauf gibt es keine aktuelle Prognose.

RULE_02:
Ein gespeicherter Vorzustand ist nur ein Vorzustand.
Er ist keine aktuelle Quelle.

RULE_03:
Wahrscheinlichkeiten dürfen nur ausgegeben werden, wenn:
- aktuelle Quellen geprüft wurden,
- Quellenqualität ausreichend ist,
- keine blockierenden Konflikte offen sind,
- Teamzustände berechnet wurden,
- Pfadsimulation abgeschlossen wurde,
- Wahrscheinlichkeiten auf 100 Prozent normalisiert wurden,
- Audit bestanden wurde.

RULE_04:
T7 ist vollständig verboten.
Externe Prognosen, Projektionen, Buchmacherquoten, Elo-/Ranking-Prognosen, FIFA-Ranking als Prognoseersatz oder Medien-Wahrscheinlichkeiten dürfen nicht gesucht, geladen, geöffnet, zitiert, gespeichert, zur Kalibrierung verwendet oder anderweitig verwertet werden.
Wenn eine Websuche solche Treffer liefert, werden sie ignoriert und nicht geöffnet.
Legacy-SRC_T7-Einträge aus älteren Prior-States dürfen nicht als aktive Quellen übernommen, aktualisiert oder ausgewertet werden.
Falls sie zur Nachvollziehbarkeit erwähnt werden müssen, dann nur als ignorierte Legacy-Information ohne Modellwirkung.

RULE_05:
Social-Media-Signale dürfen ohne Bestätigung keinen Fakt setzen.

RULE_06:
Fehlende geschlossene Premiumdaten blockieren den Lauf nicht automatisch.
Sie erzeugen Unsicherheitsflags und können die Konfidenz reduzieren.

RULE_07:
Jede finale OUTPUT-Aussage braucht eine Kausalkette:
Ursache -> Mechanismus -> Effekt -> Bedingung

RULE_08:
Am Ende müssen drei Dateien erzeugt werden:
- HTML-Ausgabe
- Prior-State-TXT für den nächsten Lauf
- kurze Änderungs-TXT

RULE_09:
Wenn Pflichtartefakte fehlen, darf keine finale Antwort ausgeliefert werden.

============================================================
3. INDEXFORMELN
============================================================

Alle Indexwerte liegen zwischen 0 und 100.

SCI = Structural Capacity Index
TBI = Tournament Body Index
CSI = Contextual Strain Index
AMCI = Adaptive Match Control Index
SAI = Shock Absorption Index
FQ = Form Quality Index

CSI ist negativ gerichtet.
Höherer CSI bedeutet höhere Belastung.

DYNAMO_STRENGTH:
D = 0.22*SCI + 0.20*TBI + 0.18*AMCI + 0.16*SAI + 0.14*FQ - 0.10*CSI

CRASH_RISK:
CrashRisk =
0.18*low_chance_creation
+ 0.17*high_transition_exposure
+ 0.15*thin_squad_risk
+ 0.14*set_piece_weakness
+ 0.13*contextual_strain_pressure
+ 0.12*psychological_fragility
+ 0.11*path_difficulty

INDEX_UPDATE:
index_new = (1 - lambda) * index_old + lambda * observed_signal

LAMBDA_BOUNDS:
min = 0.02
max = 0.45

function RUN_PIPELINE(input):

  context = INIT_RUN(input)

  model = LOAD_MODEL(context)

  prior_state = LOAD_PRIOR_STATE(input.prior_state_file)

  source_plan = BUILD_SOURCE_PLAN(model, prior_state)

  source_run = FETCH_CURRENT_SOURCES(source_plan)

  if source_run.completed == false:
      context.output_status = "NOT_CURRENT"
      output = RENDER_NOT_CURRENT(context, source_run)
      artifacts = EXPORT_ARTIFACTS(context, output, prior_state, empty_changes)
      artifact_audit = AUDIT_ARTIFACTS(artifacts)
      finalization = FINALIZE_RUN(context, artifacts, artifact_audit)
      return DELIVER(context, output, artifacts, artifact_audit, finalization)

  classified_sources = CLASSIFY_SOURCES(source_run)

  source_quality = AUDIT_SOURCE_QUALITY(classified_sources)

  bias_result = AUDIT_BIAS(classified_sources, source_quality)

  status_gate = DETERMINE_OUTPUT_STATUS(source_run, source_quality)

  if status_gate.output_status == "NOT_CURRENT":
      context.output_status = "NOT_CURRENT"
      output = RENDER_NOT_CURRENT(context, status_gate)
      artifacts = EXPORT_ARTIFACTS(context, output, prior_state, empty_changes)
      artifact_audit = AUDIT_ARTIFACTS(artifacts)
      finalization = FINALIZE_RUN(context, artifacts, artifact_audit)
      return DELIVER(context, output, artifacts, artifact_audit, finalization)

  entities = RESOLVE_ENTITIES(classified_sources, prior_state)

  signals = EXTRACT_EVENT_SIGNALS(classified_sources, entities, prior_state)

  completeness = CHECK_INPUT_COMPLETENESS(source_run, source_quality, signals)

  context.output_status = DETERMINE_OUTPUT_STATUS(source_run, source_quality, completeness)

  if context.output_status == "NOT_CURRENT":
      output = RENDER_NOT_CURRENT(context, completeness)
      artifacts = EXPORT_ARTIFACTS(context, output, prior_state, empty_changes)
      artifact_audit = AUDIT_ARTIFACTS(artifacts)
      finalization = FINALIZE_RUN(context, artifacts, artifact_audit)
      return DELIVER(context, output, artifacts, artifact_audit, finalization)

  normalized_signals = NORMALIZE_SIGNALS(signals)

  team_states = BUILD_TEAM_STATES(prior_state, entities)

  event_update_result = APPLY_EVENT_UPDATES(team_states, normalized_signals)

  team_states = RECALCULATE_INDICES(team_states, event_update_result)

  team_states = CALCULATE_DYNAMIC_STRENGTH(team_states)

  matchup_matrix = BUILD_MATCHUP_MATRIX(team_states)

  path_result = SIMULATE_PATHS(matchup_matrix, team_states)

  probability_result = NORMALIZE_PROBABILITIES(path_result)

  team_states = ATTACH_PROBABILITIES(team_states, probability_result)

  team_states = CALCULATE_CRASH_RISK(team_states, path_result)

  causal_chains = GENERATE_CAUSAL_CHAINS(team_states, signals, probability_result)

  audit = AUDIT_MODEL(
      context,
      source_run,
      source_quality,
      bias_result,
      signals,
      team_states,
      probability_result,
      causal_chains
  )

  correction_count = 0

  while audit.passed == false and correction_count < 3:
      correction = SELF_CORRECT(audit)
      APPLY_CORRECTION(correction)
      audit = AUDIT_MODEL(...)
      correction_count = correction_count + 1

  if audit.passed == false:
      context.output_status = DOWNGRADE_STATUS(audit)

  output = RENDER_OUTPUT(
      context,
      source_run,
      source_quality,
      bias_result,
      team_states,
      probability_result,
      causal_chains,
      audit
  )

  changes = BUILD_CHANGE_SUMMARY(prior_state, team_states, signals)

  artifacts = EXPORT_ARTIFACTS(context, output, team_states, probability_result, causal_chains, changes)

  artifact_audit = AUDIT_ARTIFACTS(artifacts)

  if artifact_audit.passed == false:
      artifacts = EXPORT_ARTIFACTS(context, output, team_states, probability_result, causal_chains, changes)
      artifact_audit = AUDIT_ARTIFACTS(artifacts)

  finalization = FINALIZE_RUN(context, artifacts, artifact_audit, audit)

  return DELIVER(context, output, artifacts, artifact_audit, finalization)

function INIT_RUN(input):
  required actions:
    - run_date setzen
    - run_mode setzen
    - output_status = UNKNOWN setzen
    - audit_passed = false setzen
    - artifact_audit_passed = false setzen
  returns:
    RunContext

function LOAD_MODEL(context):
  required actions:
    - Indizes laden
    - Formeln laden
    - Quellenregeln laden
    - Biasregeln laden
    - Update-Regeln laden
    - Artefaktregeln laden
  returns:
    ModelConfig

function LOAD_PRIOR_STATE(prior_state_file):
  required actions:
    - Datei prüfen
    - created_at_run_date extrahieren
    - Wahrscheinlichkeiten extrahieren
    - Indexwerte extrahieren
    - Crash-Risiken extrahieren
    - Unsicherheitsflags extrahieren
  if no prior_state_file:
    - exists = false
  returns:
    PriorState

function BUILD_SOURCE_PLAN(model, prior_state):
  required source classes:
    - T0 official tournament core
    - T1 official team/federation information
    - T3 public match and lineup data
    - T4 venue, weather, travel, altitude, rest context
    - T5 reputable news
    - T6 social warning only
  forbidden source classes:
    - T7 external forecasts, projections, bookmaker odds, Elo-/ranking-based title predictions, media probabilities
  returns:
    SourcePlan

function FETCH_CURRENT_SOURCES(source_plan):
  required actions:
    - T0 core abrufen
    - T1 team sources abrufen
    - T3 public data abrufen
    - T4 context data abrufen
    - T5 reputable news abrufen
    - T6 nur als Warnsignal prüfen
    - T7 nicht suchen, nicht laden, nicht öffnen und nicht verwenden
  returns:
    SourceRun

function CLASSIFY_SOURCES(source_run):
  required actions:
    - jede Quelle einem Tier zuordnen
    - jede Aussage als DATA markieren
    - Fakten von Interpretation trennen
    - Social Media ohne Bestätigung als warning_only markieren
  returns:
    list of SourceRecord

function AUDIT_SOURCE_QUALITY(classified_sources):
  required actions:
    - credibility_score berechnen
    - freshness_score berechnen
    - independence_score berechnen
    - specificity_score berechnen
    - actionability_score berechnen
    - total_score berechnen
    - Konflikte trennen in blocking und non_blocking
    - fehlende Daten trennen in critical und non_blocking
  returns:
    SourceQualityResult

function AUDIT_BIAS(classified_sources, source_quality):
  required checks:
    - recency bias
    - star bias
    - market bias
    - social hype bias
    - injury rumor bias
    - english language bias
    - regional undercoverage
    - closed data bias
  returns:
    BiasResult

function DETERMINE_OUTPUT_STATUS(source_run, source_quality, completeness optional):
  if source_run.completed == false:
      return NOT_CURRENT
  if source_quality.total_score < 60:
      return NOT_CURRENT
  if source_quality.unresolved_blocking_conflicts not empty:
      return NOT_CURRENT
  if source_quality.critical_missing_data not empty:
      return NOT_CURRENT
  if source_quality.total_score >= 75:
      return CURRENT
  if source_quality.total_score >= 60:
      return DEGRADED

function RESOLVE_ENTITIES(classified_sources, prior_state):
  required actions:
    - Teamnamen normalisieren
    - Spielernamen normalisieren
    - Spielorte normalisieren
    - Teamstatus prüfen
    - nicht qualifizierte Teams aus aktivem Modelluniversum entfernen
  returns:
    EntityMap

function EXTRACT_EVENT_SIGNALS(classified_sources, entities, prior_state):
  required actions:
    - nur Quellen nach prior_state.created_at_run_date als neue Events behandeln
    - Quelle -> Tier -> Datum -> Event -> Entität -> Signaltyp -> Indexwirkung erzeugen
    - Social-only-Signale begrenzen
    - T7 vollständig ausschließen; keine calibration_only-Ausnahme
    - fehlende Premiumdaten als uncertainty_flag markieren
  returns:
    list of EventSignal

function CHECK_INPUT_COMPLETENESS(source_run, source_quality, signals):
  required for CURRENT or DEGRADED:
    - aktueller offizieller Turnierkern vorhanden
    - aktuelle Team-/Availability-Daten mindestens mittel
    - aktuelle Kontextdaten mindestens mittel
    - keine blockierenden Konflikte
  returns:
    CompletenessResult

function NORMALIZE_SIGNALS(signals):
  required actions:
    - Signalwerte auf 0 bis 100 abbilden
    - Confidence setzen
    - max_model_effect setzen
    - Werte clampen
  returns:
    list of EventSignal

function BUILD_TEAM_STATES(prior_state, entities):
  if prior_state.exists:
    - TeamState aus prior_state laden
  else:
    - Initialwerte aus Quellen ableiten
    - externe Ratings maximal schwach verwenden
  returns:
    list of TeamState

function APPLY_EVENT_UPDATES(team_states, normalized_signals):
  for each signal:
    - prüfen, ob event_after_prior_state == true
    - lambda bestimmen
    - max_model_effect anwenden
    - Indexwirkung vorbereiten
  if no new model-relevant events:
    - run_result_type = CURRENT_WITH_NO_NEW_MODEL_EVENTS
    - keine künstlichen Deltas erzeugen
  returns:
    EventUpdateResult

function RECALCULATE_INDICES(team_states, event_update_result):
  required actions:
    - index_new anwenden
    - lambda bounds anwenden
    - hard caps anwenden
    - Werte clampen
  returns:
    list of TeamState

function CALCULATE_DYNAMIC_STRENGTH(team_states):
  required actions:
    - D für jedes Team berechnen
    - Haupttreiber speichern
  returns:
    list of TeamState

function BUILD_MATCHUP_MATRIX(team_states):
  required actions:
    - paarweise Matchups berechnen
    - Kontextnachteile berechnen
    - Shock-Vorteile berechnen
  returns:
    MatchupMatrix

function SIMULATE_PATHS(matchup_matrix, team_states):
  preferred:
    - exact path enumeration
  fallback:
    - seeded Monte Carlo
    - seed = 26062026
    - iterations = 100000
  returns:
    PathResult

function NORMALIZE_PROBABILITIES(path_result):
  required actions:
    - negative Werte auf 0 setzen
    - Summe auf 100 normalisieren
    - Summe prüfen: 99.5 bis 100.5
  returns:
    ProbabilityResult

function ATTACH_PROBABILITIES(team_states, probability_result):
  required actions:
    - title_probability setzen
    - final_probability setzen
    - quarter_final_probability setzen
    - early_exit_probability setzen
  returns:
    list of TeamState

function CALCULATE_CRASH_RISK(team_states, path_result):
  required actions:
    - CrashRisk pro Team berechnen
    - hohe Risiken markieren
  returns:
    list of TeamState

function GENERATE_CAUSAL_CHAINS(team_states, signals, probability_result):
  required actions:
    - für Modellführer mindestens 3 Kausalketten
    - für Top 10 mindestens 1 Kausalkette
    - für jede Änderung über 1 Prozentpunkt eine Kausalkette
    - für hohe Crash-Risiken eine negative Kausalkette
  returns:
    CausalChainSet

function AUDIT_MODEL(context, source_run, source_quality, bias_result, signals, team_states, probability_result, causal_chains):
  checks:
    - current_sources_fetched
    - t0_core_checked
    - prior_state_used_only_as_prior
    - post_prior_event_search_completed
    - all_deltas_have_new_event_chain
    - no_delta_without_post_prior_source
    - no_synthetic_delta_created
    - probability_sum_passed
    - t7_external_forecasts_blocked
    - no_t7_sources_loaded_or_used
    - social_media_sets_no_fact
    - every_output_has_causal_chain
    - confidence_not_above_data_quality
    - bias_flags_visible
  returns:
    AuditResult

function SELF_CORRECT(audit):
  if probability_sum_failed:
      return correction_target = NORMALIZE_PROBABILITIES
  if causal_chains_missing:
      return correction_target = GENERATE_CAUSAL_CHAINS
  if source_missing:
      return correction_target = FETCH_CURRENT_SOURCES
  if bad_source_usage:
      return correction_target = EXTRACT_EVENT_SIGNALS
  if bias_missing:
      return correction_target = AUDIT_BIAS
  if artifact_missing:
      return correction_target = EXPORT_ARTIFACTS

function RENDER_OUTPUT(...):
  sections:
    0 Modellbeschreibung
    1 Modellstatus
    2 Aktueller Quellenlauf
    3 Neu seit letztem Prior-State
    4 Aus Prior-State übernommen / unverändert
    5 Bias- und Unsicherheitsflags
    6 Wahrscheinlichkeitstabelle
    7 Index-Tabelle
    8 Veränderungen
    9 Kausalketten
    10 Crash-Risk
    11 Nächste Beobachtungsvariablen
    12 Audit
    13 Exportierte Dateien
  returns:
    OutputDocument

function RENDER_NOT_CURRENT(...):
  sections:
    0 Modellbeschreibung
    1 Modellstatus
    2 Warum keine aktuelle Prognose erlaubt ist
    3 Fehlende oder blockierende Quellenklassen
    4 Nächster erforderlicher Quellenlauf
    5 Audit
    6 Exportierte Dateien
  forbidden:
    - aktuelle Titelwahrscheinlichkeiten
    - aktuelles Ranking
    - aktuelle Verletzungswirkung
  returns:
    OutputDocument

function BUILD_CHANGE_SUMMARY(prior_state, team_states, signals):
  required actions:
    - nur echte Änderungen nennen
    - keine künstlichen Deltas
    - WhatsApp-kurz formulieren
  returns:
    ChangesText

function FINALIZE_RUN(context, artifacts, artifact_audit, model_audit = none):
  success criteria:
    - source_run.completed == true
    - context.output_status is CURRENT or DEGRADED
    - model_audit.passed == true when model_audit exists

      - HTML download link
      - Prior-State download link
      - Changes TXT download link
      - instruction to use Prior-State TXT for next run

GATE_CURRENT_ALLOWED:
  pass if:
    source_run.completed == true
    source_quality.total_score >= 75
    unresolved_blocking_conflicts empty
    critical_missing_data empty
    t0_core_checked == true

  pass if:
    source_run.completed == true
    source_quality.total_score >= 60
    unresolved_blocking_conflicts empty
    critical_missing_data empty

  true if:
    source_run.completed == false
    or source_quality.total_score < 60
    or unresolved_blocking_conflicts not empty
    or critical_missing_data not empty
    or t0_core_checked == false

GATE_PROBABILITIES_ALLOWED:
  pass if:
    output_status in ["CURRENT", "DEGRADED"]
    probability_result.normalization_passed == true
    probability_sum between 99.5 and 100.5
    audit.passed == true

GATE_FINAL_DELIVERY_ALLOWED:
  pass if:
    audit.passed == true
    artifact_audit.passed == true
    html download link exists
    prior-state download link exists
    changes txt download link exists

============================================================
7. OUTPUT-STATUS
============================================================

CURRENT:
  Aktuelle Quellen vorhanden.
  Quellenqualität mindestens 75.
  Keine blockierenden Datenlücken.
  Nicht blockierende Unsicherheiten sichtbar.

DEGRADED:
  Aktuelle Quellen vorhanden.
  Quellenqualität mindestens 60.
  Keine blockierenden Datenlücken.
  Konfidenz reduziert.

NOT_CURRENT:
  Aktueller Quellenlauf fehlt.
  Oder Quellenqualität unter 60.
  Oder T0-Kern fehlt.
  Oder blockierender Konflikt offen.
  Oder kritische Daten fehlen.

CURRENT_WITH_NO_NEW_MODEL_EVENTS:
  Aktuelle Quellen wurden geprüft.
  Keine belastbaren neuen modellrelevanten Events seit dem Prior-State.
  Zustand bleibt unverändert.
  Keine künstlichen Deltas.
